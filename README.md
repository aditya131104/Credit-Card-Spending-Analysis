# 💳 Credit Card Spending Behavior Analysis

## 📌 Overview
This project analyzes credit card transaction data to understand customer spending patterns, identify high-value customers, and detect seasonal trends.

## 🚀 Features
- Category-wise spending analysis
- Monthly spending trends
- Customer segmentation (Low / Medium / High)
- Interactive dashboard using Streamlit

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit

## ▶️ How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run analysis script
python analysis.py

### 3. Run dashboard
streamlit run app.py

## 📊 Dataset
Dataset used from Kaggle:  
https://www.kaggle.com/datasets/priyamchoksi/credit-card-transactions-dataset

## 📈 Key Insights
- Grocery and shopping categories generate the highest revenue  
- High-value customers contribute the most spending  
- Spending peaks during certain months  
- Customers aged 30–45 are the most active spenders  

## 📁 Output
The project generates:
- Processed dataset
- Customer segmentation file
- Graphs (PNG format)
